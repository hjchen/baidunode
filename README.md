# baidunode
express+mysql搭建的简易版移动端百度新闻
## 目录结构：
- public/index.html：百度新闻前台展示首页
- public/admin.html：百度新闻后台编辑首页
- router/index.js：后台news接口
- public/users.js：后台admin接口

## 技术栈：
- express+bootstrap+mysql+iconfont
